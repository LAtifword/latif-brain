Connection & Settings
:
Default Ollama port is 11434. For Termux on-device, use 127.0.0.1. For another device on Wi-Fi, use that device's LAN IP.
Pull once with ollama pull nomic-embed-text. Attached files are chunked, embedded, and the most relevant chunks are retrieved per question instead of dumping the whole file into context. Falls back to a raw dump automatically if this model isn't available.
Facts LATIF remembers across every chat — auto-injected into the system prompt.
Balanced: 4K context, up to 512 tokens per reply. Good mix of speed and depth on this device.
Keeping the model resident avoids the ~10–30s reload you'll otherwise hit on every message. "Forever" uses the most RAM.
Lower = faster replies. On this device, 256–512 tokens is a comfortable ceiling.
Speech recognition quality depends on your device's installed voice/TTS packs and may use your phone's on-device or Google speech engine — LATIF AI itself never sends anything to the cloud.
Opera GX-style mods restyle the whole app from a real manifest.json (Opera's public mod schema) — theme HSL, live wallpaper, WebGL shader glow, glass panels, procedural & file-based sounds. Drop a new folder in /mods/ to add your own.
Full-screen WebGL scanline/vignette/grain overlay. Auto-lowers if your device's fps drops below ~40, and falls back to a lightweight CSS scanline if WebGL isn't available.
Reads live CPU / RAM / battery / Ollama VRAM from backend/stats.py running in Termux (uvicorn stats:app --port 8000). Hidden automatically if the backend isn't reachable.
whisper.cpp gives offline Arabic/English transcription with no dependency on Google's speech engine — run backend/transcribe.py in Termux (uvicorn transcribe:app --port 8001).
