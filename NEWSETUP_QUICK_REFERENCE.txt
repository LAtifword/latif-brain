================================================================================
🚀 INSTALLATION (First Time Only - 5 Minutes)
================================================================================

1. Open Termux app on Android

2. Download or copy these scripts:
   - setup-termux-NEWSETUP.sh
   - start-latif-NEWSETUP.sh
   - configure-ollama-NEWSETUP.sh

3. Make executable:
   chmod +x ~/setup-termux-NEWSETUP.sh
   chmod +x ~/start-latif-NEWSETUP.sh
   chmod +x ~/configure-ollama-NEWSETUP.sh

4. Run setup:
   bash ~/setup-termux-NEWSETUP.sh

5. Configure Ollama (optional):
   bash ~/configure-ollama-NEWSETUP.sh

================================================================================
▶️ STARTING LATIF
================================================================================

Basic start:
  bash ~/start-latif-NEWSETUP.sh

Different port:
  LATIF_PORT=3001 bash ~/start-latif-NEWSETUP.sh

Background (keep running):
  nohup bash ~/start-latif-NEWSETUP.sh > ~/.latif/logs/latif.log 2>&1 &

================================================================================
🔌 OLLAMA CONNECTION
================================================================================

For Ollama on desktop:
  1. Note your desktop IP (Windows: ipconfig, Mac/Linux: ifconfig)
  2. Run: bash ~/configure-ollama-NEWSETUP.sh
  3. Enter desktop IP when prompted (e.g., 192.168.1.100)
  4. Port: 11434

Test connection:
  curl http://192.168.1.100:11434/api/tags

Troubleshoot:
  - Both devices must be on same WiFi
  - Check firewall allows port 11434
  - Ollama must be running on desktop

================================================================================
⚙️ CONFIGURATION
================================================================================

Edit config:
  nano /storage/emulated/0/NEWSETUP/config/android.json

Key settings:
  - llm.host: Ollama host (localhost or desktop IP)
  - llm.port: Ollama port (11434)
  - llm.model: Model name (qwen2.5:1.5b)
  - server.port: API port (3000)

For small models (low RAM):
  "model": "tinyllama:latest"

For offline (no Ollama):
  Leave default, limited functionality

================================================================================
🐛 TROUBLESHOOTING
================================================================================

NEWSETUP not found:
  ls -la /storage/emulated/0/NEWSETUP/
  If not there, extract zip to /storage/emulated/0/

Node not found:
  bash ~/setup-termux-NEWSETUP.sh

Port already in use:
  LATIF_PORT=3001 bash ~/start-latif-NEWSETUP.sh

Cannot connect to Ollama:
  curl http://DESKTOP_IP:11434/api/tags
  Check IP, firewall, WiFi connection

Out of memory:
  Edit config/android.json, use smaller model

Permission denied:
  chmod +x ~/setup-termux-NEWSETUP.sh

npm errors:
  rm -rf node_modules package-lock.json
  npm install
