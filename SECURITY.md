# Security Policy

## Reporting Security Issues

Please DO NOT open an issue for security vulnerabilities. Instead, email security@latif.dev with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Security Updates

We will release security patches as soon as possible after validation.

## Supported Versions

- Version 5.x: Fully supported
- Version 4.x and earlier: No longer supported
