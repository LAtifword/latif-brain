# knowledge-graph API

Complete documentation for the knowledge-graph API endpoint.

## Endpoints

### GET /knowledge-graph
List all knowledge-graphs.

### POST /knowledge-graph
Create a new knowledge-graph.

### GET /knowledge-graph/:id
Get a specific knowledge-graph.

### PUT /knowledge-graph/:id  
Update a knowledge-graph.

### DELETE /knowledge-graph/:id
Delete a knowledge-graph.
