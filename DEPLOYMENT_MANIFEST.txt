🚀 QUICK START ON ANDROID/TERMUX
================================================================================

1. EXTRACT THE ZIP
   mkdir -p ~/latif-v5
   cd ~/latif-v5
   unzip /path/to/latif-v5.0.0-complete.zip

2. SETUP ENVIRONMENT
   bash setup-termux.sh
   # This installs Node.js, npm, and all dependencies

3. CONFIGURE OLLAMA (Optional but recommended)
   bash configure-ollama.sh
   # Connects to Ollama on your desktop or LAN

4. START LATIF
   bash start-latif.sh
   # Server runs on http://localhost:3000

5. VERIFY
   curl http://localhost:3000/api/health
   # Should return: {"status":"ok",...}

🛠️ TROUBLESHOOTING
================================================================================

Issue: "Command not found: node"
Solution: pkg install nodejs npm

Issue: "Port 3000 already in use"
Solution: LATIF_PORT=3001 bash start-latif.sh

Issue: "Cannot connect to Ollama"
Solution: 
  1. Verify Ollama is running on desktop
  2. Check IP address: ifconfig (in Termux)
  3. Update configure-ollama.sh with correct host
  4. Test: curl http://YOUR_IP:11434/api/tags

Issue: "Out of memory"
Solution: Use smaller model in config/android.json
  "model": "tinyllama:latest"

Issue: "Database locked"
Solution: Restart LATIF (clean shutdown)
  Press Ctrl+C, wait 5 seconds, then start again

Check logs:
tail -f ~/.latif/logs/latif.log
